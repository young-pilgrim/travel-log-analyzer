# Travel Log Analyzer
Command-line travel log analyzer built in Python (JSON-based storage, modular structure).

## Features
- Add new travel logs
- Store data in JSON format
- Filter logs by category
- Calculate total expenses
- Show top categories
- Persistent data storage

## Project structure
travel-log-analyzer/
│
├── cli.py        # user interface
├── analyzer.py   # business logic and calculations
├── storage.py    # JSON read/write operations
└── data.json     # stored logs

## Technical Overview
- Python 3
- JSON storage
- Modular architecture
- CLI-based interface
